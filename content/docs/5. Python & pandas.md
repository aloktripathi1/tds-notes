### 34. `try-except-finally` — Exception Handling

#### Execution Order:

```python
# When NO exception occurs:
try → finally

# When exception occurs:
try → except → finally

# finally ALWAYS runs regardless
```

#### Full Structure:

```python
try:
    result = risky_operation()
except ValueError as e:
    print(f"Value error: {e}")
except FileNotFoundError as e:
    print(f"File not found: {e}")
except Exception as e:        # catches anything else
    print(f"Unexpected: {e}")
else:
    print("No exception occurred")  # runs only if no exception
finally:
    cleanup()                 # ALWAYS runs (close files, DB connections)
```

#### Key Rules:

- Always use **specific exception types** (not just generic `Exception`)
- `finally` is for cleanup: closing files, DB connections, releasing resources
- `else` block runs only when NO exception occurs
- ❌ `try-catch-finally` → Java syntax, not Python
- ❌ `if-else-finally` → not valid Python

### 35. Missing Value Handling

#### Detecting Missing Values:

```python
df.isnull()           # returns boolean DataFrame
df.isnull().sum()     # count NaN per column
df.isnull().sum().sum() # total NaN count
df.notnull()          # inverse of isnull()
```

#### Removing Missing Values:

```python
df.dropna()                        # drop rows with ANY NaN
df.dropna(subset=['review_text'])  # drop rows where specific column is NaN ✅
df.dropna(how='all')               # drop rows where ALL values are NaN
df.dropna(axis=1)                  # drop columns with NaN
df.dropna(thresh=3)                # keep rows with at least 3 non-NaN values
```

#### Filling Missing Values:

```python
df.fillna(0)                      # fill with 0
df.fillna('unknown')              # fill with string
df.fillna(df.mean())              # fill with column mean
df.fillna(method='ffill')         # forward fill
df.fillna(method='bfill')         # backward fill
df['col'].fillna(df['col'].median()) # fill with median
```

#### Key Rule:

- ✅ Use `dropna(subset=[...])` for targeted column NaN removal
- ❌ Replacing all NaN with "missing" → pollutes text analysis
- ❌ Deleting entire column → loses all data

### 36. `df.count()` — Count Non-Null Values

```python
df.count()          # non-null count per column ✅
df['col'].count()   # non-null count for one column

# Related:
df.notnull().sum()  # same result as df.count()
df.isnull().sum()   # count of NULL values per column
len(df)             # total rows including NaN
```

- ❌ `df.notnull()` → returns boolean, not counts
- ❌ `df.describe()` → summary stats, not just counts
- ❌ `len(df) - df.isnull()` → incorrect syntax

### 37. `column.nunique()` — Count Distinct Values

```python
df['treatment'].nunique()      # number of unique values ✅
df['treatment'].unique()       # array of unique values
df['treatment'].value_counts() # count of each unique value
df.nunique()                   # unique count for ALL columns
```

#### Comparison:

|Method|Returns|
|---|---|
|`nunique()`|Integer count of unique values|
|`unique()`|Array of unique values|
|`value_counts()`|Count of each unique value|
|`drop_duplicates()`|DataFrame with duplicates removed|

- ❌ `len(set(column))` → works but memory inefficient for large data
- ❌ `column.count()` → counts non-null values, not unique

### 38. `df.groupby().mean()` — Aggregation

```python
# Average rating per cuisine
df.groupby('cuisine')['rating'].mean()  # ✅

# Multiple aggregations
df.groupby('cuisine').agg({
    'rating': 'mean',
    'price': 'sum',
    'reviews': 'count'
})

# Multiple group-by columns
df.groupby(['cuisine', 'city'])['rating'].mean()
```

#### Common Aggregation Functions:

```python
df.groupby('col')['val'].mean()    # average
df.groupby('col')['val'].sum()     # total
df.groupby('col')['val'].count()   # count
df.groupby('col')['val'].max()     # maximum
df.groupby('col')['val'].min()     # minimum
df.groupby('col')['val'].median()  # median
df.groupby('col')['val'].std()     # standard deviation
```

- ❌ `df.sort_values('cuisine')` → sorts, doesn't aggregate
- ❌ `df.filter('cuisine')` → not valid syntax
- ❌ `df.merge('cuisine', 'rating')` → not valid syntax

### 39. `df.drop_duplicates()` — Remove Duplicates

```python
df.drop_duplicates()                    # remove fully duplicate rows
df.drop_duplicates(subset=['col'])      # based on specific column ✅
df.drop_duplicates(keep='first')        # keep first occurrence (default)
df.drop_duplicates(keep='last')         # keep last occurrence
df.drop_duplicates(keep=False)          # remove ALL duplicates

# For unique values (memory efficient):
df['treatment'].drop_duplicates().values  # ✅ most memory efficient
```

- ❌ `df['col'].unique().tolist()` → works but less memory efficient
- ❌ `list(set(df['col']))` → loses order, memory inefficient

### 40. `df.pivot_table()` + `.corr()` — Correlation Analysis

```python
# Customer similarity via purchase patterns
pivot = df.pivot_table(
    index='customer_id',
    columns='product_category',
    values='purchase_amount'
).fillna(0)

correlation = pivot.corr()  # correlation matrix ✅
```

#### Other Correlation Methods:

```python
df['col1'].corr(df['col2'])  # correlation between two columns
df.corr()                    # correlation matrix for all numeric columns
df.corr(method='pearson')    # Pearson (default)
df.corr(method='spearman')   # Spearman rank correlation
df.corr(method='kendall')    # Kendall tau correlation
```

### 41. `collections.Counter` — Counting Frequencies

```python
from collections import Counter

# Count word frequencies
words = ['apple', 'banana', 'apple', 'cherry']
Counter(words)  # {'apple': 2, 'banana': 1, 'cherry': 1}

# Count URL path segments
Counter(
    url.split('/')[2]
    for url in filtered_checkout_urls
    if len(url.split('/')) > 2
)  # ✅ product category counting

# Most common
counter.most_common(5)  # top 5 most frequent
```

### 42. Environment Variables

```python
import os

# Reading environment variables
api_key = os.getenv('API_KEY')           # returns None if not found
api_key = os.environ['API_KEY']          # raises KeyError if not found
api_key = os.getenv('API_KEY', 'default') # with default value

# Setting in terminal (Linux/Mac):
export API_KEY="your_key_here"

# Setting in terminal (Windows):
set API_KEY=your_key_here

# Using .env file with python-dotenv:
from dotenv import load_dotenv
load_dotenv()
api_key = os.getenv('API_KEY')
```

#### Why Use Environment Variables?

- Keeps secrets **out of source code**
- Never gets committed to version control
- Easy to change per environment (dev/staging/production)
- ❌ Hard-coding keys → exposed in git history
- ❌ Writing in comments → still visible in code

### 43. Seaborn — Statistical Visualization

- Built on top of Matplotlib
- Designed for **EDA and statistical visualization**
- Common plots:

```python
import seaborn as sns

sns.histplot(df['col'])              # histogram
sns.boxplot(x='cat', y='val', data=df) # box plot
sns.heatmap(df.corr())               # correlation heatmap ✅
sns.scatterplot(x='col1', y='col2', data=df) # scatter plot
sns.barplot(x='cat', y='val', data=df)  # bar plot
sns.pairplot(df)                     # pairwise relationships
sns.violinplot(x='cat', y='val', data=df) # violin plot
```

- ❌ NumPy → numerical computing, not visualization
- ❌ Django/Flask → web frameworks, not visualization
- ❌ Matplotlib → lower-level, less statistical focus

### 44. pandas — Core Operations Reference

#### Loading Data:

```python
df = pd.read_csv('file.csv')
df = pd.read_csv('file.csv', chunksize=10000)  # large files
df = pd.read_json('file.json')
df = pd.read_excel('file.xlsx')
df = pd.read_parquet('file.parquet')
```

#### Exploring Data:

```python
df.head()           # first 5 rows
df.tail()           # last 5 rows
df.shape            # (rows, columns)
df.info()           # column types and non-null counts
df.describe()       # summary statistics
df.dtypes           # data types of each column
df.columns          # column names
```

#### Filtering Data:

```python
df[df['rating'] < 3.0]                        # filter rows
df[(df['rating'] < 3) & (df['city'] == 'NY')] # multiple conditions
df[df['col'].isin(['val1', 'val2'])]           # filter by list
df[df['col'].str.contains('pattern')]          # filter by string
```

#### Sorting Data:

```python
df.sort_values('col')                    # ascending
df.sort_values('col', ascending=False)   # descending
df.sort_values(['col1', 'col2'])         # multiple columns
```

#### Adding/Modifying Columns:

```python
df['new_col'] = df['col1'] + df['col2']
df['new_col'] = df['col'].apply(lambda x: x * 2)
df['new_col'] = df['col'].str.lower()
df.rename(columns={'old': 'new'}, inplace=True)
df.drop(columns=['col1', 'col2'], inplace=True)
```

#### Merging/Joining:

```python
pd.merge(df1, df2, on='key')              # inner join
pd.merge(df1, df2, on='key', how='left')  # left join
pd.merge(df1, df2, on='key', how='outer') # outer join
pd.concat([df1, df2])                     # stack vertically
pd.concat([df1, df2], axis=1)             # stack horizontally
```

#### Saving Data:

```python
df.to_csv('output.csv', index=False)
df.to_json('output.json')
df.to_excel('output.xlsx', index=False)
df.to_parquet('output.parquet')
```
