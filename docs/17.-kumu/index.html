    17. Kumu#

    1. Kumu — Overview#

    What is Kumu?#

    • Web-based tool for creating system maps and visualizing relationships
    • Used for: network diagrams, stakeholder maps, concept maps, causal loop diagrams
    • ✅ Primary use = creating system maps and visualizing relationships — exam answer (JAN_FN Q289)
    • ❌ NOT for building financial models
    • ❌ NOT for performing real-time data analysis
    • ❌ NOT for designing geographic navigation maps

    2. Kumu — Use Cases:#

    Use CaseDescription
    System mapsShow interconnected components
    Stakeholder mapsRelationships between people/organizations
    Causal loop diagramsCause-effect relationships
    Network analysisIdentify key nodes and connections
    Impact mappingTrace outcomes and impacts
    Concept mapsLink related ideas

    3. Focus Filters — Exam Answer#

    What are Focus Filters?#

    • Highlight specific parts of visualization based on element properties

    • Show/hide connections dynamically

    • Help stakeholders focus on relevant parts of complex maps

    • ✅ Focus filters highlight specific parts of visualization based on element properties — exam answer (JAN_AN Q408)

    • ❌ Decorators → change visual appearance (size, color, shape) — different feature

    • ❌ Clustering algorithms → group elements by shared attributes — different feature

    • ❌ Color palettes → apply color schemes — not property-based highlighting

    4. Kumu Decorators:#

    • Change visual appearance of elements
    • Can modify: size, color, shape, label
    • Based on element attributes
    • Different from Focus Filters — decorators change appearance, filters highlight/hide

    5. Kumu Clustering:#

    • Groups similar elements together
    • Based on shared attributes
    • Helps simplify complex maps
    • Different from text clustering in OpenRefine — Kumu clustering is visual grouping

    6. Kumu Features — Complete Reference:#

    FeaturePurpose
